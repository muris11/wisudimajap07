# wisudimajap07

Graduation Banner Website untuk MAJAP 07 - Mahasiswa Jabodetabek Polindra

## Features

- Canva embedded video banner
- YouTube music search and player
- Responsive design for mobile and desktop
- Elegant dark green theme with gold accents

## Getting Started

```bash
npm install
npm run dev
```

## Deploy to Vercel

```bash
npm run build
vercel deploy
```

Or connect to Vercel via GitHub for automatic deployments.
